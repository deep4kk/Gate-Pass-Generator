# Gate Pass Generator

A Google Apps Script Web Application for managing employee gate passes with QR code generation and Google Docs integration for printable passes.

## Features

- **QR Code Generation**: Each gate pass generates a unique QR code for scanning
- **Google Docs Integration**: Auto-generate printable gate pass documents
- **Pass Types**: Visitor, Material, Vehicle, Emergency
- **Approval Workflow**: Security can approve/reject pass requests
- **QR Verification**: Scan QR codes to verify pass validity
- **Apple-Style UI**: Modern, responsive interface

## Setup

1. Create Google Sheet with: Users, GatePasses, AuditLog
2. Create a Google Doc template for gate passes
3. Deploy as Web App
